A universal, human-centric, replayable javascript event emitter.

Star

Installation

$ npm install noel

Basic usage

const Noel = require('Noel');
const noel = new Noel();

noel.emit('friday', arg1, arg2 ....);

noel.on('friday', partyAllNightLong).replay();
